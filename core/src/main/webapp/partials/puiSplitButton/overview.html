Description Groups a set of commands in an overlay with a default command.
Supports
  • Themeable buttons
  • Integrated with the ng-disabled attribute
  • Support for icons
  • Dynamic value using Angular Expressions
  • Button in overlay defined with <button> pui-button so that they can be moved easily into or out the widget
Known issues
  • Buttons for overlay aren't actually placed inside overlay, they are recreated. So not all attributes are used.
Planned
  • Support ng-disabled on additional buttons within overlay
Configuration The elements within pui-splitbutton which have the attribute pui-button are considered. Except for the text and ng-click attributes, no other attributes are picked up.
Name Type Default Description
icon String null Defines the icon of the button. (from theme)
iconPos String left Position of the icon, valid values are "left" and "right".
