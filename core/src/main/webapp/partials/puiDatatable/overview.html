Description Datatable displays data in tabular format. Pagination, Sorting, Selection are built-in features supporting both local and remote data.
Supports
  • Themeable table representation of data
  • Takes javascript array or function to supply data
  • Autodetection of columns based on properties of first object in array
  • Callback executed when row selected
  • Columns (properties of object) to show can be defined with nested pui-column directive
  • Sortable columns
  • Pagination of table
  • Table caption and column headers definable
  • Multiple row selection
  • Programmatic setting selected row(s) and page
Known issues
  • When you sort on a column, you loose the selection (PrimeUI code)
History
  • 0.5
    • Lazy loaded data (on pagination)
    • Support for initially sorted column
Planned
  • Integrate options of PrimeUI Paginator (like template)
Configuration
Type Default Description
Array of Object null Each object is shown on a table row where properties of first object are used as columns. No selection, sorting or pagination

Type Default Description
function (callback) null Result of function should be an array of objects that make up the table data. First object properties are used as columns. No selection, sorting or pagination
When directive points to an object
Name Type Default Description
tableData Array of objects / function(callback) null Data to shown as an array of objects or function that supplies (remotely) the info.
columns Array of String / Array of Object null Information about the columns to shown. See below
rowSelect function(event, data) null When specified, it activates the selection possibilities and function is called when user selects a row in the table.
rowUnselect function(event, data) null When specified, it is called when user deselects a row in the table. No effect when selection options is not activated (rowSelect or selectionMode specified)
selectionMode String null Determines if only one (value single) or multiple rows can be selected. If rowSelect function specified and no value for this option, the default of 'single' is taken
paginatorRows int null When specified, it activates the paginator possibilities and it shows the data data with n rows at a time
selectedData Array of int null Array containing the indices of the selected rows in the data table. When value is programmatic changed, visual row selection is updated on screen. It takes into account the selectionMode specified and works in combination with the user selection.
selectedPage int null Only valid in combination with paginatorRows option. It contains the index (0 based) of the selected page in the paginator. When value is programmatic changed, the page selection on screen is also updated (and thus showing other set of rows)
lazy (0.5) boolean false When data is loaded lazy, the tableData property should point to a function that loads the data lazy into the browser.
totalRecords (0.5) int null When lazy loading is in effect, the total number of rows needs to be specified upfront in order that the paginator can work correctly.
sortField (0.5) String null The name of the property / column on which the initial sorting (at first display) must be performed.
sortOrder (0.5) String null The order for the initial sorting where the values 'down' and 'up' are recognized.

column definition

Column information isn't needed but when specified, this is the where the information is taken from.
  1. Child elements with pui-column directive on
  2. columns property of the object referred to in the pui-datatable directive
  3. Autodetect from the first object in the data

pui-column directive

Type Default Description
String null Name of the property of the object to show. It will also be the name of the column title. No sorting options.
When directive points to an object
Name Type Default Description
field String null Required, name of the property to show in this column
headerText String null Name of column header. When not specified, no header text is displayed
sortable boolean false Is the data in the table sortable by this column.
