Description Extends standard Checkbox with skinning capabilities.
Supports
  • Integrated with checked attribute, when present it take precedence over the controller value.
  • Integrated with ng-disabled attribute
  • Integrated with ng-change so feedback possible when user selects or deselects checkbox
  • Integrated ng-true-value and ng-false-value, scope value can be a user defined String
  • High level directive puiCheckbox for easier creation of box with label.
  • Integration with ng-show and ng-hide only with pui-checkbox (no workaround needed)
Known issues
  • ng-checked is not supported
History
  • 0.4
    • High level directive puiCheckbox the include box and label
    • Support for ng-show and ng-hide only with puiCheckbox
  • 0.6
    • Widget is now also available as element/tag pui-input
