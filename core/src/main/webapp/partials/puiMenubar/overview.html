Description Shows menu in a bar layout.
Supports
  • Themeable menu
  • Support for sub menu
Known issues
  • When individual JavaScript files are used, the file for puiMenu is also needed.
Planned
  • puiMenuItem directive for uniformization and easier encoding of menu
  • ng-disabled support on puiMenuItem
  • Menu 'model' that allows creation of menu based on JavaScript objects
Configuration
Name Type Default Description
autoDisplay boolean true Are submenus shown when hovered (true) or after a click (false)?
