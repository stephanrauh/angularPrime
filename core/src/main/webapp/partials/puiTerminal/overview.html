Description Terminal is a text based interface, optionally linked to a backend.
Supports
  • Sending commands to javascript handler method
  • Customize prompt message
  • Optional welcome message
Known issues
  • The clearTerminal function works only on the last widget in the scope/controller.
Planned
Configuration The value of the pui-terminal directive can have 2 types:
  • function (command) to process the command entered by the user
  • Object, scope defined, containing the options and the handler method
Name Type Default Description
terminalHandler function( String) Function needs to return the response based on the command entered on the screen
prompt String prime $ Prompt message shown before the command entry
welcomeMessage String First line in the terminal widget with the welcome message
The directive adds also a new function clearTerminal to the scope. This function can be used to clear the output of the terminal widget.
