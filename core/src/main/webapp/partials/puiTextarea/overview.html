Description Extension to standard text area with skinning capabilities, autoresize, autocomplete and maximum number of characters.
Supports
  • Auto resize when more rows are needed
  • Integration with ngDisabled
  • Autocomplete of words
  • Limit the number of characters and show the remaining count
Known issues
Planned
Configuration
Name Type Default Description
autoResize Boolean false Defines if textarea increase its size automatically.
maxlength Integer null Determines the maximum letters in textarea.
display string null JQuery selector that determines where the template is shown
template String '{0}' Template the string which shows, how many letters left.
minQueryLength Integer 3 Number of characters to be typed to run a query.
queryDelay Integer 700 Delay in ms before sending each query.
pui-autocomplete directive can be used for autocomplete of words. It must point to a function (request, response) in the scope.
