Description Advanced widget to select an item from a collection.
Supports
  • Customizable widget by themes
  • Multiple selection
  • Integration with ng-disabled attribute
  • Programmatic selection of the selected option
  • Feedback on user selection
  • Custom content
Known issues
History
  • 0.6
    • Custom content can be defined with pui-content widget
Planned
  • Support for option definition within code.
  • Dynamic options
  • Programmatic selection in case of multiple selections
Configuration
Name Type Default Description
scrollHeight integer 200 Maximum height boundary of overlay panel in pixels, if total height of items exceed this value, scrollbar is displayed.
callback function(selectedValue) null Callback functional which is called when item selection is made.
The custom content can be defined by using an element pui-content nested within the pui-listbox element/tag.
The content can be any valid HTML where you can use the placeholders %LABEL% and %VALUE%.
