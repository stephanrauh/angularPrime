Description Used to reorder items between differents lists.
Supports
  • Themeable selection of options
  • Selection by buttons, double click and drag-drop
  • Integration with ng-disabled directive
  • Programmatic selection of options
  • Options defined in Javascript object
  • options can be filtered
  • Reorderable options
  • Custom content
Known issues
  • When individual JavaScript files are used, the file for puiButton is also needed.
Planned
  • Allow dynamic number of options
Configuration
Type Default Description
Array of String/Object null When string, it is used as value and label for each option. When it is an object, then properties value and label are used.
When directive points to an object
Name Type Default Description
filter boolean false When set to true, filtering fields are shown above the source and target lists
filterMatchMode String startsWith Indicate which filtering mode will be used. Supported values are startsWith, contains, endsWith, custom
filterFunction function(value, filter) null Filter function to use in case filterMatchMode is custom. This function gets item label and filter value as parameters and required to return a boolean value to implement filtering. true accepts the item and false rejects it.
sourceCaption String null Caption text for source list.
targetCaption String null Caption text for target list.
showSourceControls boolean false Show the controls for reordering the items in the source list.
showTargetControls boolean false Show the controls for reordering the items in the target list.
sourceData Array of String/Object null When string, it is used as value and label for each option. When it is an object, then properties value and label are used.
callback function (eventType, value) null Callback function called when item is selected or deselected from the target list. eventType is ADD or REMOVE, value is the value of the option.
The custom content can be defined by using an element pui-content nested within the pui-picklist element/tag.
The content can be any valid HTML where you can use the placeholders %LABEL% and %VALUE%.
