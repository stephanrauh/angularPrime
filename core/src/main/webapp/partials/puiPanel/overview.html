Description Grouping component providing with content toggle and close features.
Supports
  • Toggling and closing can be specified in the view or in the controller
  • Title attribute supports AngularJS expressions
  • Collapsing and expanding panel can be triggered programmatic
  • Toggle orientation and speed and closing speed can be configured.
Known issues
Planned
  • Callbacks called for triggers collapse/expand/close
Configuration
Name Type Default Description
collapsed Boolean undefined If specified, content of panel can be expanded and collapsed and determines the initial state. When option is specified in the controller, changing the controller value changes the screen appearance.
closable Boolean false Defines if content of panel can be hidden using close option (irreversible action).
toggleOrientation String vertical Defines orientation of toggle direction, valid values are "vertical" and "horizontal".
toggleDuration Number/String normal Defines the duration of the toggle effect, valid values are "slow", "normal", "fast" or a number in milliseconds.
closeDuration Number/String normal Defines the duration of the close effect, valid values are "slow", "normal", "fast" or a number in milliseconds.
