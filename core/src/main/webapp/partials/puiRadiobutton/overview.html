Description Extends standard radio button with skinning capabilities.
Supports
  • Themeable radio button
  • Label can be click to change selection
  • Integration with ng-disabled, ng-change and ng-repeat
  • Programmatic setting the selected option
  • High level directive puiRadiobutton for easier creation of radio thick with label.
  • Integration with ng-show and ng-hide only with pui-radiobutton (no workaround needed)
Known issues
  • Without the name attribute, you don't have the single selection in a 'group'
  • Specifying label in combination with ng-repeat doesn't work.
History
  • 0.4
    • High level directive puiRadiobutton the include thick box and label
    • Support for ng-show and ng-hide only with puiRadiobutton
Planned
