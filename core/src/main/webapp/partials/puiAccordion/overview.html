Description Accordion is a container component with vertically stacked panels.
Supports
  • Single panel or multiple panels can be expanded
  • Programmatic setting the expanded panel, not supported in case of multiple expandable panels
  • Callback function available when user has changed the expanded panel
  • Panel can be in an included file.
  • Support for changing number of panels dynamically
  • puiPanel can be used to define the panel
Known issues
  • Title of panel cannot contain an AngularJS expression (see dynamic example)
History
  • 0.3
    • Support for accordion panels from included files
    • Number of panels can be changed dynamically
    • puiPanel can be used to define the panel
Planned
  • Lazy loading
Configuration
Type Default Description
Array of String null URL's of the included files as panel
When directive points to an object
Name Type Default Description
activeIndex Integer/Array 0 Index of the active tab. If multiple is enabled, an array of active indexes.
multiple Boolean false Defines if multiple tabs can be active at the same time.
callback function (index) null Callback called when the user selects a panel (becomes expanded).
urls Array of String null URL's of the included files as panel
