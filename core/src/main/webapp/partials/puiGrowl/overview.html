Description The Mac's growl widget
Supports
  • Show a skinnable message on the right top for a certain time.
  • Messages can be made sticky
Known issues
  • Icon images must be specified in application
			.pui-growl-image {
			    background: url("../demo/images/messages.png") no-repeat;
			}
History
  • 0.3
    • puiGrowl is now also a service
    • div with id growl no longer needed in HTML
    • Support for sticky messages.
    • New messages don't remove current message. function clear exposed to remove remaining messages
