Description Theme switcher
Supports
  • Shows dropdown for theme selection
  • Configurable feedback for storing selected theme
  • Theme change without page reload
Known issues
Planned
  • Customizable set of themes
  • Customizable contents of dropdown panel
  • Customizable directory of themes
Configuration
Name Type Default Description
name String aristo Default theme when page is shown
dropdown boolean null Is a dropdown shown for theme selection?
Themes can be set programmatic by calling themeService.setSelectedTheme(String) function.
