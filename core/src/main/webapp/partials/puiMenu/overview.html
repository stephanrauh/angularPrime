Description Highly customizable navigation/command component that supports dynamic and static positioning.
Supports
  • Themeable menu
  • Shown after button click or always visible
  • items can be 'grouped' under a title
  • Support for sub menu
  • Support for context menu (right click)
Known issues
  • When context menu is bound to the page, it stays available on all pages of the application
Planned
  • Customization of location where menu is shown when used as popup
  • Customization of trigger that shows popup menu (now always click)
  • puiMenuItem directive for uniformization and easier encoding of menu
  • ng-disabled support on puiMenuItem
  • Menu 'model' that allows creation of menu based on JavaScript objects
Configuration
Name Type Default Description Comment
target string JQuery selector (like #id) of element that activates the popup menu.
autoDisplay boolean true Are submenus shown when hovered (true) or after a click (false)? Ignored when isContextMenu or isSlideMenu set to true or no sub menu items
isContextMenu boolean false Is this menu used as a context menu (right click)? Can't be combined with isSlideMenu.
isSlideMenu boolean false Are the sub menus shown by sliding them in? Can't be combined with isContextMenu
