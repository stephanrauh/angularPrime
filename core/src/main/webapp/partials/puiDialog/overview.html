Description A container component that can overlay other elements on page. Dialog has several customization options such as modal, resize, width and height.
Supports
  • Modal an non modal overlay
  • Show and hide effects
  • Close when 'esc' is pressed
Known issues
  • effectOptions, rtl, appendTo and buttons from PrimeUI widget is not supported
Planned
Configuration
Name Type Default Description
draggable Boolean true Enables dragging using header.
resizable Boolean false Enables content resizing.
location Boolean center Defines the position of the dialog, valid values are combination of "left", "right", "top", "bottom" like "left right" or absolute coordinates like "50,150".
width int 300 Width of the dialog.
height int auto Height of the dialog.
minWidth int 150 Minimum width of a resizable dialog.
minHeight int 25 Minimum height of a resizable dialog.
modal boolean true Defines if background should be blocked.
showEffect string fade Effect to run to show the dialog.
hideEffect string fade Effect to run to hide the dialog.
effectSpeed string normal Speed of the animation in milliseconds or predefined values like slow, normal and fast.
closeOnEscape boolean true Specifies if pressing escape key should hide the dialog.
closable boolean true Adds a close icon to the header to hide the dialog.
minimizable boolean false Adds a toggle icon to the header to minimize the dialog.
maximizable boolean false Adds a toggle icon to the header to maximize the dialog.
