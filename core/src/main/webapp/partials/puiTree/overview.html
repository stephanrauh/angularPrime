Description Displays hierarchical data or for navigation purposes.
Supports
  • Declarative tree node definition or based on JavaScript object
  • Tree nodes can be loaded from remote call
  • In lazy load mode, child nodes are retrieved when node is expanded
  • Possibility to define icon in front of node label
  • Single and multiple selection modes
  • Callback functions for selecting and unselecting a node
  • Slide animation when node is expanded/collapsed
Known issues
Planned
  • Support for programmatic collapse / expand of nodes.
Configuration
Type Default Description
function(event, ui) null Node selection function. ui.data contains the 'id' of the selected node.
Name Type Default Description
nodes Array of objects / function(ui, callback) null Tree data to display. For the object structure see the pui-treenode description below. When the lazy remote mode is active, the ui.data contains the 'id' of the expanded node.
selectionMode String null The selection mode for the nodes. When a nodeSelect function is defined, and this property is not specified, a default value of 'single' is assumed. The other allowed value is 'multiple'.
nodeSelect function(event, ui) null Fired when a node is selected. ui.data contains the 'id', ui.node contains the li DOM element.
nodeUnselect function(event, ui) null Fired when a node is unselected. ui.data contains the 'id', ui.node contains the li DOM element.
lazy boolean false In lazy mode, children nodes are loaded on demand. Prefer this option when dealing with large datasets. Requires function form for the nodes attribute.
icons String/Object null Defines icon sets for TreeNode types. When it is a String, it defines the CSS class of the icon like ui-icon-image. When it is an object, the expanded and collapsed attribute defines the CSS class for the icon in the corresponding state of the node.
animate boolean false Displays a slide animation when nodes are expanded/collapsed when enabled.

node definition

When elements are found with the pui-treenode directive defined, these are taken for tree build-up and information specified in the nodes property of the pui-tree directive is ignored.
Name Type Default Description
label String null Label of the node.
iconType String null Type of the icon which needs to be displayed on the node. See also icons property of the pui-tree directive.
data String null 'id' of the node used in the selection and deselection events.
When data is not specified, the id attribute is taken from the element. If this is also not specified, the label property is used as a default.
