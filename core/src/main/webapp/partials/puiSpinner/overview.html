Description Input component to provide a numerical input via increment and decrement buttons.
Supports
  • Integrated with theme
  • Support for keyboard and mouse wheel
  • Integrated with ng-disabled and ng-change
  • Prefix and postfix can be defined
Known issues
  • When using prefix/suffix, it is only shown when you leave the field.
Planned
Configuration
Name Type Default Description
step Double 1.0 Step factor to increment/decrement.
min Double undefined Minimum value when using the decrement button
max Double undefined Maximum value when using the increment button
prefix String undefined Prefix for the value when blur event occurs
suffix String undefined Suffix for the value when blur event occurs
