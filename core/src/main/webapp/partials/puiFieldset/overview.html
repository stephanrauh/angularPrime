Description Grouping component which also provides content toggling.
Supports
  • Grouping of content skinned according to the theme
  • Optional toggleable content
  • Expression allowed in legend tag and within body
  • Programmatic toggleable
  • Callback function available when user toggles the content
Known issues
  • No warning when callback set but not the collapsed option.
Planned
Configuration
Name Type Default Description
toggleDuration Number/String normal Defines the duration of the toggle effect, valid values are "slow", "normal", "fast" or a number in milliseconds.
collapsed Boolean undefined When set, defines the display status of the fieldset and marks is as toggleable.
callback function() undefined Allows to react when the user changes the collapsed state of the content.
